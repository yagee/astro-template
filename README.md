# Astro Template

## Install

```sh
pnpm i
```

## Development

```sh
pnpm dev
```

## Upgrade

```sh
pnpm dlx @astrojs/upgrade
```

## Check using prettier

```sh
pnpm prettier
```

## Fix using prettier

```sh
pnpm prettier:fix
```

## Check using eslint

```sh
pnpm lint
```

## Build

```sh
pnpm build
```

## Optimize using jampack

```sh
pnpm optimize
```

## Deploy

```sh
netlify deploy --prod -d dist
```
